## 🚀 시작하기

### 필요한 파일 준비
- **엑셀 파일** (.xlsx, .xls, .csv)
- 첫 번째 열(A열)에 참가자 이름이 입력되어 있어야 함
- 첫 번째 행은 제목으로 무시됩니다

### 엑셀 파일 예시
| 이름   | 기타정보 |
|--------|----------|
| 홍길동 | 21학번   |
| 김철수 | 22학번   |
| 이영희 | 21학번   |

## 📖 기본 사용법

### 1단계: 참가자 명단 업로드
1. **"📁 엑셀 파일 선택"** 버튼 클릭
2. 준비한 엑셀 파일 선택
3. 성공 메시지와 함께 참가자 수 확인

> 💡 **팁**: 파일 업로드 후 "○명의 참가자를 불러왔습니다!" 메시지가 나타나면 성공

### 2단계: 좌석 구성 설정
1. **"각 열별 좌석 수"** 입력란에 좌석 배치 입력
2. 앞열부터 순서대로 쉼표(,)로 구분하여 입력
3. 예시: `12,14,16,18,20,20,20,10`

#### 좌석 구성 예시
- `10,12,14` → 1열 10석, 2열 12석, 3열 14석 (총 36석)
- `15,15,15,15` → 각 열 15석씩 4열 (총 60석)
- `8,10,12,14,16` → 단계적으로 증가하는 구성 (총 60석)

### 3단계: 자리 배치 실행
1. **"🎯 자리 배치 시작"** 버튼 클릭
2. 자동으로 랜덤 배치 완료
3. 결과 화면에서 배치 상황 확인

## 🎯 우선 배정 기능

특정 인원을 원하는 자리에 미리 배정할 수 있습니다.

### 우선 배정 설정 방법
1. **"3. 우선 배정"** 섹션에서 정보 입력:
   - **이름**: 참가자 명단에 있는 정확한 이름
   - **열**: 배치할 열 번호 (1부터 시작)
   - **번호**: 해당 열의 좌석 번호 (1부터 시작)

2. **"+ 우선 배정 추가"** 버튼으로 여러 명 설정 가능

3. **"삭제"** 버튼으로 잘못 입력한 항목 제거

### 우선 배정 예시
| 이름   | 열 | 번호 | 설명           |
|--------|----|------|----------------|
| 홍길동 | 1  | 5    | 1열 5번 자리   |
| 김철수 | 2  | 1    | 2열 1번 자리   |
| 이영희 | 3  | 10   | 3열 10번 자리  |

### 주의사항
- ⚠️ 참가자 명단에 없는 이름은 무시됩니다
- ⚠️ 좌석 범위를 벗어나는 자리는 무시됩니다
- ⚠️ 중복된 자리는 먼저 입력된 것만 적용됩니다

## 🔄 자리 변경 기능

랜덤 배치 후 수동으로 자리를 바꿀 수 있습니다.

### 자리 변경 방법
1. **"🔄 자리 변경 모드"** 버튼 클릭
2. 모드가 활성화되면 버튼이 **"❌ 자리 변경 모드 종료"**로 변경
3. 바꾸고 싶은 **첫 번째 자리** 클릭 (빨간색으로 선택됨)
4. 바꾸고 싶은 **두 번째 자리** 클릭
5. 자동으로 두 자리가 교환됨

### 자리 변경 예시
```
변경 전: 홍길동(1-5), 김철수(2-3)
↓ 1-5 클릭 → 2-3 클릭
변경 후: 김철수(1-5), 홍길동(2-3)
```

### 자리 변경 팁
- 🟦 **일반 배정**: 보라색 좌석
- 🟧 **우선 배정**: 주황색 좌석
- 🟥 **선택된 좌석**: 빨간색 좌석
- ⚪ **빈 좌석**: 흰색 좌석 (선택 불가)

## 📊 결과 내보내기

### 엑셀 파일로 내보내기
1. **"📊 엑셀 내보내기"** 버튼 클릭
2. `자리배치결과_YYYY-MM-DD.xlsx` 파일 다운로드
3. 엑셀 파일 내용:
   - **열**: 좌석이 위치한 열 번호
   - **좌석번호**: 해당 열의 좌석 번호
   - **이름**: 배정된 참가자 이름
   - **구분**: 우선배정 또는 일반배정

### 엑셀 결과 예시
| 열   | 좌석번호 | 이름   | 구분     |
|------|----------|--------|----------|
| 1열  | 5번      | 홍길동 | 우선배정 |
| 2열  | 1번      | 김철수 | 일반배정 |
| 2열  | 2번      | 이영희 | 일반배정 |

## 🔧 문제 해결

### 자주 발생하는 문제들

#### 1. 파일 업로드 오류
**증상**: "파일 읽기 오류가 발생했습니다" 메시지
**해결법**:
- 엑셀 파일 형식 확인 (.xlsx, .xls, .csv)
- 첫 번째 열에 이름이 정확히 입력되어 있는지 확인
- 파일이 손상되지 않았는지 확인

#### 2. 좌석 부족 오류
**증상**: "참가자 수가 좌석 수보다 많습니다" 메시지
**해결법**:
- 좌석 구성에서 좌석 수를 늘리기
- 예시: `10,12,14` → `12,14,16,18`

#### 3. 우선 배정이 적용 안 됨
**해결법**:
- 이름이 참가자 명단과 정확히 일치하는지 확인
- 열과 좌석 번호가 설정 범위 내인지 확인
- 중복 자리 설정이 없는지 확인

#### 4. 자리 변경이 안 됨
**해결법**:
- 자리 변경 모드가 활성화되어 있는지 확인
- 빈 자리가 아닌 배정된 자리를 클릭했는지 확인
- 브라우저를 새로고침 후 다시 시도

### 권장 워크플로우
```

```
